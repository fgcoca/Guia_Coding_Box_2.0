## <FONT COLOR=#007575>**8. Sensor RFID**</font>

!!! info ""
    !!! Note "Aviso importante"
        En el momento de la elaboración de este tutorial para Coding Bos 2.0, no existe ningún bloque disponible en STEAMakersBlocks que permita la integración o el uso directo del sensor RFID I2C. Por este motivo, las funcionalidades relacionadas con este dispositivo no pueden implementarse mediante bloques estándar de STEAMakersBlocks para ESP32 y requerirían métodos alternativos de programación o futuras actualizaciones de la plataforma que incorporen soporte para este sensor.
